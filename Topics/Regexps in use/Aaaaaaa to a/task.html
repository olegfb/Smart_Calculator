Aaaaaaa to a

Write a program that reads one string, then replaces all repeated "a" letters in it with one "a" ("aaAAaaa" must be replaced with one "a" character), and prints the resulting text. Your code must be case insensitive.

See examples for better understanding.

Input: text: String

Output: newText: String


Sample Input:
And I will aalwaaaaaays love you. I will Aaalways love you.

Sample Output:
and I will always love you. I will always love you.


Sample Input:
Mamaaa, life had just begun, But now I've gone and thrown it all awaaaaAAAay

Sample Output:
Mama, life had just begun, But now I've gone and thrown it all away



Memory limit: 256 MB
Time limit: 8 seconds
